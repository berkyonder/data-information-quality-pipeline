# Data Directory

This folder is intentionally kept without the original CSV dataset.

## Why data files are missing
The artists/artworks CSV files used in the course project are not available for redistribution in this public portfolio repository.

## Local usage
If you have legitimate local access to the original files, place them in this `data/` directory (or adjust paths in `notebooks/DIQ.ipynb` accordingly).

Expected format: CSV files used by the original notebook workflow.

> Note: Do not commit raw data files to this repository.
