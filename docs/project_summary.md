# Project Summary

## Title
Data Information Quality Pipeline for Artists and Artworks Data

## Purpose
This project applies practical data quality methods to a real academic dataset context, with emphasis on diagnostics and data reliability improvements.

## Core Methods
- Data profiling and quality assessment
- Missing-value analysis and treatment
- Value normalization and standardization
- Outlier detection
- Duplicate/near-duplicate detection

## Deliverables
- Executable notebook workflow (`notebooks/DIQ.ipynb`)
- Written report (`reports/report.pdf`)
- Portfolio documentation for reproducibility and project context

## Data Availability Note
Original CSV data is not included in this repository.
